### DOME 3 installation

(0) rename C:\dome3 to C:\dome3-old, if there is an old version.
(1) unzip dome3.zip to C:\dome3
(2) unzip j2sdk1.4.2_13.zip to C:\j2sdk1.4.2_13
(3) to execute DOME client, double-click C:\dome3\scripts\RunDome.bat
    to execute DOME server, double-click C:\dome3\scripts\RunServer.bat
