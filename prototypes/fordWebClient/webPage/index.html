DOME Web Portal Example
The DOME software prototype is the result of 7 years of research on integrated simulation at MIT. This site has been created to illustrate how DOME can be used through a web interface. Several integrated models from the Windnoise pilot study can be accessed from this page.

Type your username and password below. Click on blue links to open models directly. Use the start dome client button if you would like work within the DOME client application. DOME help is available within the client.
